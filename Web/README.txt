Para abrir el servidor node, utilizar el siguiente comando en la terminal:

npm start

IMPORTANTE: Hay que ubicarse en la terminal en el directorio Web

Una vez hecho esto: se puede abrir la web en localhost:8080

El pipeline gstreamer está integrado a través de una conexión tcp con el servidor de nodeJs.
A este pipeline se le han añadido efectos de video que se aprecian a la hora de streamear.
